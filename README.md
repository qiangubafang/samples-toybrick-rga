# samples-toybrick-rga
RGA sample


这是RK3399 老版本的rga示例； 而RK3399平台最高支持到linux 4.4 内核（如英飞凌 OK3399-C）； Android7.1.2系统； <br/>

如果你恰好在找RK3399平台上瑞芯微的RGA示例，那么恭喜你，这一版可以编译运行。<br/>

使用RK最新的RGA和MPP仓库代码，在RK3588上运行正常。 在RK3399上无法运行；<br/>

踩了很多坑才找到这个仓库，留存给不幸要在此平台上开发小伙伴。 <br/>


对应的老版本仓库地址： <br/>
